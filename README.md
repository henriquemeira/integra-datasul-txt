# Conversor Datasul TXT → JSON

Single Page Application (React + Tailwind) para converter arquivos posicionais (Datasul) em JSON.

Funcionalidades:
- Upload por arrastar/soltar ou input
- Visualização das linhas originais
- Parser que monta a hierarquia (Tipo 1 -> Itens Tipo 2 -> Extras Tipo 8 e Duplicatas Tipo 4)
- Download do JSON formatado

Como rodar:
1. npm install
2. npm run dev

Arquivos de layout e exemplo em `doc/`.
